# 💰 Expense Tracker (Full Stack)

A modern full-stack Expense Tracker web application built with React and Node.js. It helps users manage their income and expenses, view their financial summary, and analyze spending through interactive charts.

## 🚀 Features

* ✅ Add transactions
* ✅ Edit transactions
* ✅ Delete transactions
* ✅ Search transactions
* ✅ Filter by Income/Expense
* ✅ Dashboard with Balance, Income & Expense summary
* ✅ Pie Chart visualization
* ✅ Responsive design (Mobile & Desktop)
* ✅ Loading state
* ✅ Custom 404 Page
* ✅ REST API backend

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/Expense-Tracker-FullStack.git
```

### Install Frontend

```bash
cd client
npm install
npm run dev
```

### Install Backend

```bash
cd ../server
npm install
npm run dev
```

## 🌐 API Endpoints

| Method | Endpoint                | Description           |
| ------ | ----------------------- | --------------------- |
| GET    | `/api/transactions`     | Get all transactions  |
| POST   | `/api/transactions`     | Add a new transaction |
| PUT    | `/api/transactions/:id` | Update a transaction  |
| DELETE | `/api/transactions/:id` | Delete a transaction  |
